Berzerk Arcade for the Tang Nano 9K Dev FPGA Board. Pinballwiz.org 2025

Notes:
Controls are PS2 keyboard, see readme file in tools folder for instructions.
Consult the schematics folder for details regarding keyboard - audio - VGA connections.

Build:
* Obtain correct roms file for berzerk, see make berzerk proms script in tools folder for rom filenames.
* Unzip rom files to tools folder.
* Run the make berzerk proms script in the tools folder.
* Place the generated prom files inside the proms folder.
* Open the berzerk_tn9 project file using Gowin and compile.
* Program Tang Nano 9K Board.
